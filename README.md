# 🤖 AI Powered Exam Preparation & Personalized Study Assistant

## 📌 Overview
This project is a full-stack web application designed to help students prepare for exams efficiently using AI-based personalized learning techniques.

It provides quizzes, performance tracking, and customized study plans based on user performance.

---

## 🚀 Features

- 📝 Dynamic Quiz System (Subject & Difficulty based)
- 📊 Performance Tracking & Score Analysis
- 🎯 Personalized Study Plan Generation
- ⏱ Timer-based Mock Exams
- 📚 Study Session Tracking
- 🔐 User Authentication (Login / Signup)

---

## 🛠 Tech Stack

### 💻 Frontend
- React.js (TypeScript)
- Tailwind CSS
- shadcn/ui
- React Router DOM

### ☁️ Backend / Database
- Supabase (PostgreSQL)
- Supabase Auth

## 🔐 Environment Variables

Create a `.env` file in root:
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/AI-Exam-Preparation-Assistant.git

2.Navigate to project folder:
cd AI-Exam-Preparation-Assistant

3. Install dependencies:
npm install

4. npm install
npm run dev
